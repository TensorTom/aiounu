
# unu

An asyncio module for [unu](https://u.nu/) in Python3 using aiohttp. Forked from 

## Request

        pip install requests  

## Install

        pip install aiounu
        
## Import

        from unu import unu  

## Use

        await unu(url="https://vcinex.com", action="shorturl", format="simple",keyword="")

Only the "url" variable is necessary.
